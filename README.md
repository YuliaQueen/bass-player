# Bass Tabs Player

[![CI](https://github.com/YuliaQueen/bass-player/actions/workflows/ci.yml/badge.svg)](https://github.com/YuliaQueen/bass-player/actions/workflows/ci.yml)

Локальное веб-приложение для разучивания басовых партий из Guitar Pro
(`.gp/.gp3-8/.gpx`) и MusicXML (`.xml/.musicxml/.mxl`) файлов.
Партитура отображается горизонтальной лентой
нот + табов, играет MIDI, бежит курсор по текущей доле, страница сама
скроллится. Скорость, метроном, loop секции, сохранение настроек на каждый
трек — то, что нужно для практики.

## Запуск

Нужен **Docker Desktop**. Локальный PHP/Composer не требуется — бэк работает
через [Laravel Sail](https://laravel.com/docs/13.x/sail).

```bash
# Бэк (Laravel 13 + Postgres 18 в Docker)
cd backend && ./vendor/bin/sail up -d
cd ..

# Фронт (Vite на хосте)
npm install
npm run dev
```

- **Frontend:** http://localhost:5173 (Vite проксит /api и /tabs на бэк)
- **Backend:** http://localhost:8001
- **Postgres:** localhost:5432

Полезные алиасы из Sail:

```bash
sail artisan migrate          # миграции
sail test                     # Pest-тесты
sail bin pint                 # форматтер PHP
sail composer require ...     # composer внутри Sail
```

## Откуда брать файлы

**Guitar Pro:**

- **[Songsterr](https://www.songsterr.com/) Plus** → у любой песни «Download
  Guitar Pro tab»
- Любой другой источник: `.gp`, `.gp3`, `.gp4`, `.gp5`, `.gpx`, `.gp7`, `.gp8`

**MusicXML:**

- **[MuseScore.com](https://musescore.com/)** → экспорт в MusicXML
- Любой нотный редактор (Sibelius, Finale, Dorico) — есть экспорт в `.musicxml`
- `.mxl` — сжатый MusicXML (zip-архив с .xml внутри)

⚠️ **Поддержка MusicXML в alphaTab экспериментальная.** Часть файлов
рендерится без звука, или с октавными ошибками (особенно при
`<clef-octave-change>`). **Workaround:** в MuseScore выбери
**Файл → Экспорт → Guitar Pro 5 (.gp5)** — это формат «первого класса»
для alphaTab, играет всегда стабильно.

Лимит размера загрузки — 25 МБ.

## Как добавить песню

Любым из двух способов:

1. **Drag & drop** на окно браузера — overlay подскажет «Отпустите для
   загрузки», файл сохранится в `backend/storage/app/tabs/` и автоматически
   откроется
2. **Кнопка `+ Файл`** в сайдбаре — стандартный file picker
3. **Положить руками** в `backend/storage/app/tabs/` через Finder, потом
   обновить страницу — файл появится в списке

Удаление — крестик `✕` справа от файла в сайдбаре (с подтверждением).

## Управление

### Кнопки в панели контролов

- **▶ Play / ⏸ Pause** — старт/пауза
- **■ Stop** — пауза с возвратом в начало
- **Скорость** — слайдер 0.25x–1.5x (alphaTab сам делает time-stretch без
  изменения тона)
- **Громкость** — общая громкость 0–100%
- **Метроном** — громкость метронома 0–100% (по умолчанию 0)
- **Count-in** — отсчёт перед началом воспроизведения
- **Loop трек** — зацикливание всей песни
- **Loop секции** — зациклить указанный диапазон тактов

### Loop секции

- Введи в поля `от` / `до` номера тактов и нажми `⟳`
- **Или** просто **выдели мышкой по партитуре** — зажми на одном такте, тяни до
  другого, отпусти. Loop применится автоматически
- `×` — сбросить секцию

### Прогресс-бар (тонкая полоса под контролами)

- Заполняется по мере воспроизведения
- Клик по полосе — перемотка в нужную позицию

### Hotkeys (когда фокус не в текстовом поле)

- `Space` — Play/Pause
- `←` / `→` — на 1 такт назад/вперёд
- `↑` / `↓` — скорость +/-0.05
- `Home` — в начало

### Сохранение настроек

На каждый файл сохраняются: скорость, громкости, метроном, count-in, loop-режим
с диапазоном, последняя позиция воспроизведения. Хранятся в `localStorage`
браузера. При следующем открытии файла всё восстановится.

При перезагрузке страницы открывается последний выбранный файл.

## Известные ограничения / quirks

- **Кириллица в rehearsal-метках** (названия секций «1-й Куплет», «Припев» и
  т.д.) — alphaTab по дефолту использует шрифт без кириллицы; перебито
  на Arial через `elementFonts`. Дефис в «1-й» иногда сливается визуально
  с буквой «й» — артефакт SVG-рендеринга, не баг данных
- **Загруженные файлы лежат в `backend/storage/app/tabs/`** — содержимое
  gitignored, переустановка `node_modules` или `vendor/` их не тронет
- **Single-user режим** — пока общая библиотека на установку. Multi-user
  с auth добавится в следующей фазе (Laravel Sanctum SPA)
- **Несколько дорожек в файле** — показывается только басовая (MIDI program
  32–39 General MIDI). Если несколько басовых — берётся первая

## Smoke test

После первой установки или больших правок:

1. Открой http://localhost:5173/
2. Выбери файл из библиотеки → должна нарисоваться партитура
3. Нажми Play → играет MIDI, курсор бежит, лента скроллится
4. Поставь скорость 0.5x → слышно замедление, тон тот же
5. Выдели мышью такты 5–10 → должен зациклиться этот кусок
6. Перезагрузи страницу → открывается тот же файл, скорость 0.5x восстановилась,
   loop секции восстановился
